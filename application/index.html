UK Traffic Accidents Database

About the data

The UK government amassed traffic data from 2000 and 2016, recording over 1.6 million accidents in the process and making this one of the most comprehensive traffic data sets out there. It's a huge picture of a country undergoing change. Note that all the contained accident data comes from police reports, so this data does not include minor incidents. This set includes accidents from 2012-2014.

For more information, please see the credited source in the footer. For more information on attributes, visit the UK Bureau of Transportation.

Accident reports are comprehensive, detailing severity, response to, and environment surrounding collision.

About the application

Explore UK government-collected traffic accidents data from 2012 to 2014. Report an accident, delete old reports, search, get insights about road junctions, and plan around dangerous travel dates.
